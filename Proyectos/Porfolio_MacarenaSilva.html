Landing Page

Este proyecto consiste en una página de aterrizaje moderna y visualmente atractiva para una agencia de viajes llamada TravelNow. El objetivo fue crear una experiencia sencilla y elegante que mostrara los destinos más populares, una breve presentación de la empresa, y un formulario de contacto básico. La estructura se diseñó únicamente con HTML5, empleando etiquetas semánticas para mejorar la organización del contenido y facilitar la accesibilidad. El diseño fue completamente responsivo gracias a media queries en CSS3, adaptándose perfectamente a dispositivos móviles, tablets y escritorio. Uno de los aspectos más llamativos es el header con imagen de fondo y efecto overlay, que ofrece una bienvenida visual impactante. También se incorporaron tarjetas con sombra (box-shadow) para destacar los destinos, así como animaciones suaves con transition y hover. El formulario de contacto, aunque simple, fue diseñado con atención a la usabilidad, incluyendo :focus y :placeholder personalizados para mejorar la experiencia del usuario. En resumen, es un proyecto limpio, elegante y funcional que demuestra la capacidad de construir páginas profesionales usando solo HTML y CSS.

Blog

El blog Code & Coffee es un sitio web de estilo minimalista diseñado para compartir artículos y reflexiones sobre desarrollo web. Hecho exclusivamente con HTML y CSS, el foco del proyecto es la legibilidad, la simplicidad visual y la organización clara del contenido. Se implementó una estructura con header, nav, main y footer, separando cada sección del blog para facilitar la navegación. El diseño está basado en una maquetación con flexbox, que permite reorganizar los artículos en columnas en pantallas grandes y en una sola columna en dispositivos móviles. Cada entrada del blog se presenta en una tarjeta con título, fecha, categoría y un breve resumen. El estilo tipográfico fue cuidadosamente seleccionado para favorecer la lectura, con títulos grandes y márgenes generosos. También se diseñó un menú de navegación lateral (oculto en móviles), que permite acceder rápidamente a las distintas categorías del blog. Los estilos incluyen paletas en tonos grises con acentos en azul pastel, logrando un ambiente tranquilo y profesional. Este proyecto muestra cómo crear una estructura de blog limpia, funcional y adaptable con HTML y CSS, ideal para principiantes o como base para desarrollos futuros.

Web Responsive

La Galería ArtVibe es un sitio web estático creado exclusivamente con HTML y CSS, que muestra una colección de obras de arte contemporáneo en formato digital. El reto era presentar una experiencia visual envolvente, sin utilizar JavaScript, asegurando que la navegación fuera clara y atractiva desde cualquier dispositivo. El diseño se basa en una rejilla fluida (CSS Grid), que adapta la galería de imágenes automáticamente según el tamaño de la pantalla. Cada obra se presenta dentro de una tarjeta con bordes redondeados, bordes suaves y transiciones al pasar el ratón. Se utilizó object-fit para mantener las proporciones de las imágenes, garantizando armonía visual sin distorsiones. Se añadieron efectos sutiles como hover:scale y cambio de opacidad para dar dinamismo, además de un encabezado tipo banner que presenta el nombre de la galería y una breve introducción. El sitio también cuenta con un pie de página fijo con datos de contacto y redes sociales, estilizado con flexbox. Este proyecto demuestra cómo combinar diseño visual con estructura semántica usando solamente HTML y CSS, ideal para artistas, diseñadores o pequeños estudios que deseen tener presencia online sin complejidad técnica.
